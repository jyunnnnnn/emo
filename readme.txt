
EMO App是一個專為打造淨零綠生活而產生的平台。透過此平台，用戶可以紀錄每日的減碳行為，並及時檢視其對環境的影響。
我們希望能藉此提高民眾環保意識，共同打造綠色未來。


系統特色
EMO App 綜合多種元素，包括視覺化紀錄、環保理念，並為用戶帶來趣味和互動。主要特點如下：
1.減碳項目設計：針對日常生活習慣（如飲食、交通）設計，讓用戶輕鬆落實環保。隨著紀錄增加，用戶能感受到日常行為對環境的影響，體會到自己的貢獻，提高參與意願。
2.數據/視覺化呈現：結合Google Map，將位置信息與減碳紀錄視覺化標記，讓用戶清晰看到自己的環保行為。數據化呈現用戶的貢獻，使其更直接地了解自己的環保成效。
3.友善介面：簡單易懂的操作方式，跨平台功能，提升使用體驗，吸引更多人參與環保行動。
4.趣味性功能：成就系統、社群功能和排行榜，增加使用樂趣和成就感，激勵用戶積極參與環保活動。
5.可擴充性：管理員角色可定制和改進App，系統設計便於未來更新和維護，提升可擴充性和可靠性。



提升流程品質或系統品質的措施:
1.code review :
	-code至少有2個人看過
	-code可讀性
	-註解 
	-變數命名有沒有意義
	-code功能是否正常，符合預期
	-code至少要有1個人看得懂
2.安全性:
	-https協議加密
	-AES演算法加密個資
3.定義資料庫schema:
	-Primary key
	-required
	-Not Null
	-資料型態
4.merge或遇到conflict時，舉行線上會議討論
5.git commit message規範:
	-feat: 新增功能
	-modify:修改功能
	-fix:修復bug
	-style:網頁格式以及code格式修改
6.branch命名規定: 
	- branch開發者名稱 + 開發日期 +時間　ex: Jyun 11/21 18:42
	- 一個人最多兩個分支，一個最新的正在改的，一個上一版的
	-一個功能就commit一次
